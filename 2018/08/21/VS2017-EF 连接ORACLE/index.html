VS2017-EF 连接ORACLE

发表于 2018-08-21
字数统计: 204 | 阅读时长 ≈ 1

1、在你项目Model层中nuget安装选中项

  1. 下载oracle，这个有点麻烦~就不具体说咯，这里就说下已经下好oracle的情况下，还要下一个oracle for vs2017 Tools。就是我们的驱动。

http://www.oracle.com/technetwork/topics/dotnet/downloads/odacmsidownloadvs2017-3806459.html

下载之后，直接安装即可（注意，安装前最好将VS先退出）：

打开vs2017，测试一下新建的时候能不能选oracle数据源， –ODP.NET模式

如果没有,请执行下一步：

找到第一步的安装目录，例如我本机的：C:\Program Files (x86)\Oracle Developer Tools for VS2017\odt

运行 目录下的：Oracle.VsDevTools.15.0，重新安装插件，再打开vs2017就可以了。
